# Projects

These are a list of a couple projects I have worked on, some on my own, and some from courses I have taken.

# CIS White Paper:
This paper explains how Apple Inc. can improve its cybersecurity using CIS Controls by strengthening data protection, running regular penetration tests, and training employees to recognize threats. It emphasizes that even a highly secure company must continuously improve to protect user data and maintain trust.

# CTI Report:
This report analyzes the cyber threat group APT29 and a fictional attack on a state government using phishing and malware to gain access and steal sensitive information. It explains how threat intelligence tools like OSINT, the MITRE ATT&CK Framework, and indicators of compromise help understand the attacker's behavior and identify security gaps. This report also states immediate, mid-term, and long-term defense strategies, emphasizing that continuous monitoring, stronger policies, and smart threat intelligence are important to prevent future attacks and protect government systems.

# SnakeGame:
This is an in class project that I made in a data structures class using c++. This is a very simple snake game that uses linked lists as a snake. Since I didn't know how to use a GUI in c++, I used printing as the game board. This also only works on windows as it uses windows.h. It constantly deletes and updates the board causing it to look like the snake is moving.

# StockTrackerPython:
This is a project I made on my own for fun, it is pretty simple but thought it would be fun to do. It asks the user to input 5 or less stocks, as well as the date the user started investing. It then asks the user for each date and amount of money put in, then calculates the total return from all the investments to the current date.
